# loan-analysis-power-BI-dashboard
##  Project Overview
Interactive Power BI dashboard to analyze loan 
sanctioned and disbursed amounts and support 
loan recovery strategy.

##  Tools Used
- Power BI (DAX, Drill-down filters)
- Excel (Data Source)

##  Dashboard Features
- KPI Cards (245K, 212.73K, 195.71K, 104.20K)
- Funnel chart showing loan pipeline
- Trend analysis (2017-2019)
- Product wise loan breakdown
- Channel distribution pie chart

##  Key Results
- Improved reporting transparency
- Enabled leadership to drill into loan segments
- Supported accurate resource allocation
